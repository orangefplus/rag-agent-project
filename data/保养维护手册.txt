车辆保养维护手册

一、保养周期总则
车辆保养分为常规保养和定期更换两大类，严格按照保养周期执行可延长车辆使用寿命，保障行车安全。智能客服系统会根据行驶里程和时间自动提醒保养。

二、常规保养周期表

1. 首次保养（免费）
- 时间：购车后6个月或5000公里（以先到为准）
- 内容：全面检查、软件升级、底盘紧固
- 注意：必须在规定时间内完成，否则影响质保

2. 常规保养（每10000公里或12个月）
- 更换机油机滤（混动车型）
- 检查制动系统、悬挂系统
- 检查轮胎磨损和胎压
- 检查冷却液、制动液液位
- 检查12V蓄电池状态
- 软件系统检查升级

3. 中度保养（每20000公里）
- 包含常规保养全部内容
- 更换空调滤芯
- 更换空气滤芯
- 检查火花塞（混动车型）
- 检查传动皮带
- 轮胎换位

4. 大保养（每40000公里）
- 包含中度保养全部内容
- 更换制动液
- 更换冷却液
- 更换变速箱油
- 检查高压系统绝缘
- 更换减速器油

三、易损件更换周期

1. 机油及机滤
- 混动车型：每10000公里或12个月
- 机油规格：0W-20全合成机油，约4.5L
- 机滤：每次换机油同步更换

2. 空调滤芯
- 更换周期：每20000公里或每年（雾霾严重地区建议半年）
- 作用：过滤进入车内的空气，保障PM2.5净化效果
- 症状：空调风量变小、有异味时需提前更换

3. 空气滤芯
- 更换周期：每20000公里
- 作用：过滤进入发动机的空气（混动车型）
- 症状：加速无力、油耗增加时检查

4. 制动液
- 更换周期：每40000公里或2年
- 规格：DOT4制动液
- 症状：制动变软、制动距离变长时检查

5. 冷却液
- 更换周期：每40000公里或4年
- 规格：长效有机酸型冷却液（OAT）
- 症状：液位低报警、水温高时检查

6. 轮胎
- 更换周期：60000-80000公里或4-6年
- 磨损极限：胎纹深度小于1.6mm必须更换
- 检查：每月检查胎压，每10000公里换位
- 规格：前后轮标准胎压2.5bar，满载2.7bar

7. 雨刮片
- 更换周期：每年或刮不干净时
- 症状：刮水有异响、留有水痕

8. 12V蓄电池
- 更换周期：3-5年
- 症状：启动困难、电压低于12V
- 规格：12V 60Ah AGM蓄电池

9. 制动片
- 更换周期：前轮约40000公里，后轮约60000公里
- 症状：刹车异响、制动距离变长
- 检查：每次保养检查厚度，小于3mm需更换

四、新能源专属保养

1. 动力电池
- 日常：避免经常充到100%或用到0%，建议保持在20%-80%
- 快充：频繁快充会加速电池衰减，建议每周至少一次慢充
- 温度：高温暴晒后不要立即充电，低温充电前预热电池
- 检查：每年到店做电池健康度检测（SOH）
- 质保：8年或15万公里，衰减超过30%可质保更换

2. 高压系统
- 检查周期：每年或20000公里
- 检查内容：绝缘电阻、高压线束、接插件
- 注意：高压系统维修必须由专业人员操作

3. 充电口
- 保养：定期清洁充电口，检查有无烧蚀痕迹
- 注意：充电口有水渍时不要充电

4. 减速器油
- 更换周期：首次20000公里，之后每40000公里
- 规格：75W-90齿轮油

五、季节性保养

1. 春季保养
- 检查空调系统，更换空调滤芯
- 检查雨刮系统，更换老化雨刮片
- 底盘防锈处理
- 检查天窗排水孔

2. 夏季保养
- 检查冷却系统，确保散热良好
- 检查轮胎胎压，高温适当降低
- 检查空调制冷效果
- 避免暴晒，使用遮阳挡
- 检查蓄电池，高温加速老化

3. 秋季保养
- 检查暖风系统
- 更换防冻冷却液（到期）
- 检查雨刮和玻璃水（换防冻型）
- 检查轮胎，准备应对低温

4. 冬季保养
- 使用冬季玻璃水（-25度以下地区）
- 检查蓄电池状态，低温性能下降
- 胎压适当调高0.1-0.2bar
- 电池预热：充电前和出行前预热电池
- 检查暖风和座椅加热功能

六、日常检查项目

1. 每次出车前
- 检查轮胎外观和胎压
- 检查灯光是否正常
- 检查仪表盘有无报警
- 检查续航里程是否充足

2. 每周检查
- 检查各种液位（冷却液、玻璃水、制动液）
- 检查轮胎胎压
- 清理车内外杂物

3. 每月检查
- 检查轮胎磨损情况
- 检查雨刮片
- 检查空调滤芯
- 清洁车辆

七、保养预约指南
1. 预约方式：可通过语音"预约保养"、手机APP、客服电话预约
2. 预约信息：车牌号、行驶里程、保养类型、期望时间、门店选择
3. 预约优势：到店免等待、工时费9折、赠送洗车
4. 保养时长：常规保养约1.5小时，大保养约3-4小时
5. 保养费用：常规保养约500-800元，大保养约2000-3500元

八、保养注意事项
1. 必须使用原厂配件和指定规格油液
2. 保养记录影响质保权益，请保留完整记录
3. 非授权店保养可能影响质保
4. 软件升级建议在保养时同步完成
5. 保养后首次行驶注意检查有无异常
