汽车维修与常见故障处理手册

一、发动机系统维修

1. 发动机无法启动
故障原因排查：
- 蓄电池电量不足：用万用表测量蓄电池电压，低于12V需充电或更换
- 启动机故障：听是否有"咔咔"声，有则启动机碳刷磨损
- 燃油泵不工作：打开点火钥匙听油箱是否有"嗡嗡"声，无则油泵故障
- 点火线圈故障：拔出火花塞帽检查是否有火花
- 曲轴位置传感器故障：用OBD诊断仪读取故障码
维修方法：
- 蓄电池：搭电启动后怠速充电30分钟，电压持续低于12V需更换
- 启动机：拆解更换碳刷或整体更换启动机总成
- 燃油泵：更换燃油泵总成，注意清洗油箱
- 点火线圈：更换对应缸的点火线圈

2. 发动机异响
异响类型与诊断：
- "嗒嗒"声（冷车明显）：气门间隙过大，需调整气门间隙
- "铛铛"金属敲击声：连杆轴承磨损，需拆解发动机更换轴承
- "嘶嘶"声：进气歧管漏气，检查密封垫
- "哗哗"声：正时链条松弛，需更换正时链条组件
- "吱吱"尖叫声：皮带打滑或张紧轮失效，更换皮带和张紧轮
诊断步骤：
1. 用听诊器确定异响位置
2. 区分冷车/热车、怠速/加速时的异响变化
3. 逐缸断火法确定故障缸
4. 用OBD读取数据流辅助判断

3. 发动机抖动
常见原因：
- 个别缸不工作：火花塞积碳/损坏、点火线圈失效、喷油嘴堵塞
- 进气系统漏气：真空管破裂、进气歧管垫片老化
- 燃油系统：燃油压力不足、喷油嘴雾化不良
- 机械故障：气门积碳、活塞环磨损导致缸压不足
维修方法：
- 逐缸断火找出不工作缸
- 更换火花塞（铱金火花塞4万公里，镍合金2万公里）
- 清洗喷油嘴（超声波清洗）
- 清洗进气门积碳（核桃砂清洗或化学清洗）

4. 发动机过热
原因排查：
- 冷却液不足：检查膨胀罐液位，查找泄漏点
- 水泵故障：检查水泵轮是否转动，叶轮是否腐蚀
- 节温器故障：节温器打不开，小循环无法切换大循环
- 散热器堵塞：外部虫尸/柳絮堵塞，内部水垢堵塞
- 电子扇不转：检查风扇继电器、温控开关、风扇电机
紧急处理：
- 立即靠边停车，不要立即熄火（怠速散热）
- 不要打开膨胀罐盖（高温蒸汽烫伤）
- 等温度降至90度以下再检查冷却液

二、变速箱维修

1. 自动变速箱换挡顿挫
原因：
- 变速箱油变质/不足：油液变黑、有焦糊味
- 电磁阀故障：换挡电磁阀卡滞
- 离合器片磨损：摩擦片烧蚀
- 阀体故障：液压控制阀卡滞
维修方法：
- 更换变速箱油（每6万公里，使用原厂指定规格）
- 清洗阀体或更换电磁阀
- 更换离合器片组件（大修）
- 变速箱电脑匹配学习

2. 手动变速箱挂挡困难
原因：
- 离合器分离不彻底：离合器片磨损、分离轴承故障
- 同步器磨损：换挡时齿轮打齿
- 变速箱油不足/变质
- 换挡拉线调整不当
维修方法：
- 更换离合器三件套（压盘、摩擦片、分离轴承）
- 更换同步器组件
- 更换变速箱油（GL-4 75W-90）
- 调整换挡拉线

三、制动系统维修

1. 刹车异响
- 尖叫异响：刹车片磨损到极限（报警铁片摩擦刹车盘），需立即更换刹车片
- 闷响：刹车盘变形或不平，需光盘或更换
- 金属摩擦声：刹车片完全磨光，铁磨铁，需更换刹车片和刹车盘
- 雨天异响：刹车片受潮，轻踩几脚即可消除

2. 刹车软/制动力不足
原因排查：
- 制动液不足/变质：液位低于MIN线，或含水量超过3%
- 制动系统有空气：需排气
- 制动总泵/分泵漏油：检查各轮制动分泵
- 真空助力器故障：踩刹车时感觉不到助力
- ABS系统故障：ABS泵或轮速传感器问题
维修方法：
- 更换制动液（DOT4，每4万公里或2年）
- 制动系统排气（从远到近：右后→左后→右前→左前）
- 更换漏油的分泵/总泵
- 更换真空助力器

3. 手刹/电子手刹故障
- 手刹拉线松弛：调整拉线螺母或更换拉线
- 电子手刹无法释放：检查电子手刹开关、执行电机
- 电子手刹异响：执行电机齿轮磨损

四、电气系统维修

1. 蓄电池频繁亏电
排查步骤：
- 测量静态电流：锁车后等待10分钟，电流应小于50mA
- 常见漏电原因：加装设备（行车记录仪接常电）、模块不休眠、门锁开关故障
- 蓄电池健康度：用蓄电池检测仪测量CCA值和内阻
维修方法：
- 排除漏电设备
- 更换蓄电池（AGM蓄电池寿命3-5年）
- 检查发电机充电电压（怠速时13.8-14.4V正常）

2. 发电机不充电
原因：
- 发电机皮带断裂/松弛
- 发电机碳刷磨损
- 调节器故障
- 发电机定子/转子线圈烧毁
维修方法：
- 更换发电机皮带（检查张紧器）
- 更换碳刷和调节器（可单独更换）
- 更换发电机总成

3. 各种传感器故障
- 氧传感器故障：油耗增加、排放超标、发动机灯亮，需更换（前氧8万公里，后氧10万公里）
- 空气流量计故障：加速无力、怠速不稳，清洗或更换
- 节气门脏污：怠速抖动、易熄火，清洗节气门（每2万公里）
- 水温传感器故障：风扇不转或常转、冷车启动困难
- 轮速传感器故障：ABS灯亮、车速表不准，清洁或更换

五、底盘维修

1. 悬挂异响
- 过减速带"咯噔"声：下摆臂球头磨损、平衡杆胶套老化
- 转弯"嘎吱"声：减震器上座轴承损坏
- 颠簸路"咚咚"声：减震器漏油失效
维修方法：
- 更换下摆臂球头或总成
- 更换平衡杆胶套和开口胶套
- 更换减震器总成（成对更换）

2. 转向系统
- 转向沉重：转向助力油不足、转向助力泵故障、电子助力电机故障
- 转向异响：转向机齿轮齿条磨损、转向柱万向节缺油
- 跑偏：四轮定位不准、轮胎气压不均、悬挂部件变形
维修方法：
- 检查/补充转向助力油
- 更换转向助力泵
- 做四轮定位（参数：前束、外倾角、后倾角）

3. 轮胎异常磨损
- 胎面中间磨损：胎压过高
- 胎面两侧磨损：胎压过低
- 胎面单侧磨损：四轮定位不准
- 胎面波浪状磨损：减震器失效或轮胎动平衡不良
- 胎面杯状磨损：悬挂球头松旷
维修方法：
- 调整胎压至标准值
- 做四轮定位
- 做轮胎动平衡
- 更换磨损严重的轮胎

六、空调系统维修

1. 空调不制冷详细排查
第一步：检查空调压缩机是否工作
- 打开AC，观察压缩机皮带轮是否吸合
- 不吸合：检查空调继电器、压力开关、压缩机离合器

第二步：检查制冷剂压力
- 低压侧0.15-0.25MPa，高压侧1.3-1.7MPa为正常
- 压力过低：制冷剂泄漏，需查漏
- 压力过高：冷凝器散热不良或制冷剂过多

第三步：查漏方法
- 电子检漏仪检测
- 荧光检漏（加荧光剂运行后用紫外灯检查）
- 肥皂水检漏（接口处涂抹）

第四步：常见泄漏点
- 冷凝器（石击损伤）
- 空调管路接口O型圈
- 蒸发器（腐蚀穿孔）
- 压缩机轴封

2. 空调有异味
原因：蒸发器表面霉菌滋生
处理方法：
- 更换空调滤芯（活性炭型）
- 清洗蒸发器（专用空调清洗剂喷入进风口）
- 使用空调管道杀菌除味剂
- 日常预防：到目的地前5分钟关AC开风量，吹干蒸发器

七、新能源车专属维修

1. 动力电池维修
- 电池均衡：单体电压差异>0.1V需均衡充电
- 电池包漏液：立即断电，专业拆解检查
- 电池温度异常：检查温度传感器、冷却液循环
- SOH下降过快：检查充电习惯、BMS数据
注意：动力电池维修必须由持证高压电工操作

2. 电机控制器故障
- IGBT模块过热：检查冷却系统
- 驱动板故障：更换驱动板
- 通信故障：检查CAN总线连接
维修方法：电机控制器通常整体更换，不建议拆解维修

3. 充电系统故障
- 慢充不充：检查充电枪、车载充电机(OBC)、充电口温度传感器
- 快充不充：检查BMS通信、充电桩协议、高压继电器
- 充电跳枪：充电口温度过高、电池温度过低、电压异常
维修方法：
- 更换车载充电机(OBC)
- 更换充电口总成
- BMS软件升级

4. 高压系统绝缘故障
- 绝缘电阻低于100Ω/V：高压系统漏电
- 排查：逐段断开高压部件测量绝缘
- 常见漏电点：电机、电池包、高压线束、空调压缩机
注意：绝缘故障必须专业维修，禁止自行拆解

八、维修工具与设备
1. 基础工具：套筒扳手套装、扭矩扳手、卡环钳、轴承拉马
2. 诊断设备：OBD诊断仪（读取故障码和数据流）、万用表、示波器
3. 专用工具：发动机吊架、变速箱千斤顶、四轮定位仪、轮胎拆装机
4. 新能源专用：绝缘手套(1000V)、绝缘工具套装、高压验电笔、绝缘电阻测试仪

九、维修安全注意事项
1. 维修前必须断电（断开蓄电池负极，新能源车断高压）
2. 举升车辆确认支撑点牢固
3. 使用扭矩扳手按规定力矩紧固螺栓
4. 制动系统维修后必须排气
5. 新能源车高压系统维修前等待5分钟（电容放电）
6. 维修完成后做功能测试和路试
