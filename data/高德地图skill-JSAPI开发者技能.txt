高德地图 JS API SKILL（开发者技能）
最后更新时间：2026年06月18日
来源：https://lbs.amap.com/api/skill/developer-skills/jsapi-skills

一、产品介绍
高德地图 JSAPI Skills 是一套专为 AI IDE 设计的 AI 编程技能包。它将高德地图 JavaScript API v2.0 的官方文档、最佳实践和代码模板整合为结构化的技能文件，使 Cursor、Claude、Cline 的 AI Coding 工具能够：
- 精准理解高德地图 API 的使用方法
- 自动生成符合官方规范的地图代码
- 主动避免常见的开发陷阱和安全问题
- 提供经过验证的完整代码示例

二、产品特点
1. 精准的 API 知识：基于高德地图 JSAPI v2.0 官方文档构建，覆盖完整的 API 能力，确保生成的代码符合最新规范。
2. 安全优先：内置安全配置指南，自动提醒开发者配置安全密钥，避免因鉴权问题导致地图白屏。生产环境推荐代理方案，保护您的密钥安全。
3. 模块化设计：按功能模块组织文档（地图控制、覆盖物、图层、LBS服务），AI 可按需引用，精准回答问题。
4. 代码可验证：所有代码示例均经过验证，AI 会自我校验生成代码的可用性，确保输出可以正常运行。

三、能力介绍
1. 地图控制
- 地图初始化：异步加载、3D/2D 视图、地图样式配置
- 视图交互：缩放、平移、俯仰角、旋转控制
- 生命周期管理：加载完成回调、资源销毁释放
- 地图控件：比例尺、工具条、鹰眼、图层切换

2. 覆盖物绘制
- 点标记 (Marker)：基础标记、自定义图标、DOM 内容
- 海量标注 (LabelMarker)：支持万级点位、文字图标避让
- 矢量图形：折线、多边形、圆形、矩形
- 信息窗体 (InfoWindow)：默认信息窗、自定义内容窗体
- 右键菜单：自定义地图/覆盖物右键交互

3. 图层管理
- 标准图层：标准地图、卫星图、路网图
- 3D 图层：建筑楼块、室内地图
- 自有数据图层：Canvas 图层、WMS/WMTS 服务

4. LBS 服务与插件
- 地理编码：地址转坐标、坐标转地址
- 路径规划：驾车、步行、骑行、公交路线
- POI 搜索：周边搜索、关键词搜索、输入提示
- 事件系统：点击、拖拽、缩放等交互事件

四、快速接入
第一步：获取 Skills 文件
- 方式一（推荐）：git clone https://github.com/AMap-Web/amap-skills；cd jsapi-skills
- 方式二：直接下载 https://a.amap.com/jsapi_demos/static/amap-jsapi-skills/amap-jsapi-skills-v1.1.0.zip 最新版本压缩包并解压

第二步：在 Cursor 中配置 Skill
Cursor 支持通过 .cursor/skills/ 目录加载自定义技能。
2.1 创建 Skills 目录：mkdir -p .cursor/skills
2.2 链接或复制 Skill 文件
- 方式一（软链接，推荐，方便更新）：
  macOS / Linux：ln -s /path/to/jsapi-skills/amap-jsapi-skill .cursor/skills/amap-jsapi-skill
  Windows（管理员 CMD）：mklink /D .cursor\skills\amap-jsapi-skill C:\path\to\jsapi-skills\amap-jsapi-skill
- 方式二（直接复制）：cp -r /path/to/jsapi-skills/amap-jsapi-skill .cursor/skills/

2.3 项目目录结构
your-project/
├── .cursor/
│ └── skills/
│   └── amap-jsapi-skill/
│     ├── SKILL.md
│     └── references/
│       ├── map-init.md
│       ├── marker.md
│       ├── security.md
│       └── ...
├── src/
├── package.json

第三步：验证配置
1) 打开 Cursor IDE
2) 打开您的项目
3) 按 Cmd/Ctrl + L 打开 AI Chat
4) 输入测试问题，例如：
"帮使用高德 JSAPI 开发一个 POI 搜索与路径规划应用：0. 地图ak在env.js文件中，使用这个作为地图ak；1. 起点搜索；2. 分类选择；3. POI 列表；4. POI 预览；5. 路径规划（默认驾车，支持切换步行/公交/骑行）。"
如果 AI 在思考过程能正确使用高德地图 JSAPI 的 skill 文件并生成包含安全配置的完整代码，说明 Skill 已成功加载。

第四步：申请高德开发者 Key
1) 访问高德开放平台控制台 https://console.amap.com/ 注册并登录
2) 进入「应用管理」→「我的应用」
3) 点击「创建新应用」
4) 添加 Key，选择「Web端(JS API)」服务
5) 获取 Key 和安全密钥
重要：自 2021 年 12 月起，高德地图 JSAPI v2.0 强制要求配置安全密钥，否则地图将无法加载。

六、最佳实践
1. 安全配置
// 开发环境：直接配置（仅用于本地调试）
window._AMapSecurityConfig = {
  securityJsCode: '您的安全密钥',
};
// 生产环境：使用代理转发（推荐）
window._AMapSecurityConfig = {
  serviceHost: 'https://your-domain.com/_AMapService',
};

2. 资源释放
// 组件卸载时务必销毁地图，防止内存泄漏
useEffect(() => {
  // 初始化地图...
  return () => {
    map?.destroy();
  };
}, []);

3. 按需加载插件
// 只加载需要的插件，减少首屏资源体积
AMapLoader.load({
  key: '您的Key',
  version: '2.0',
  plugins: ['AMap.Scale', 'AMap.Geocoder'],
});

八、常见问题
Q1: 地图显示白屏或报错 INVALID_USER_KEY
A: 请检查：
1) 是否在 AMapLoader.load 之前配置了 window._AMapSecurityConfig
2) Key 和安全密钥是否匹配
3) Key 是否已在高德控制台启用

Q2: Cursor AI 没有使用 Skill 中的知识
A: 请确保：
1) Skill 文件放置在 .cursor/skills/ 目录下
2) SKILL.md 文件存在且格式正确
3) 尝试在提问时明确提及"高德地图"或"AMap"

Q3: 如何更新 Skill？
A: 如果使用复制方式，需要重新复制最新文件。
